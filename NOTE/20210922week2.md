# 第二週筆記
## 紅黑樹
* 紅黑樹（英語：Red–black tree）是一種自平衡二元搜尋樹，是在電腦科學中用到的一種資料結構，典型用途是實現關聯陣列。
  1. 節點是紅色或黑色。
  2. 根是黑色。
  3. 所有葉子都是黑色（葉子是NIL節點）。
  4. 每個紅色節點必須有兩個黑色的子節點。（從每個葉子到根的所有路徑上不能有兩個連續的紅色節點。）
  5. 從任一節點到其每個葉子的所有簡單路徑都包含相同數目的黑色節點。
* BIG O(log(n))
